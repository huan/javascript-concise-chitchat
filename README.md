## Usage

TL;DR:

```shell
npm install
npm start
```

### 1 Install

```shell
npm install
```

### 2 Train

```shell
npm run train -- --batch_size 32 --epochs 3 --latent_dim 64 --num_samples 10000 dist/
```

### 3 Chat

```shell
npm run chat
```
